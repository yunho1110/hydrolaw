# HydroLaw-AI 예상 질의 모음

업종별 전형적 질의 + 테스트 케이스 (수치형·맥락형·복합형·음성형)

## 세차장 (세차 / 자동차 세차 시설)

### Q1 (수치형 — 정확 조회)
**질의**: "세차장 BOD 기준이 뭔가요?"  
**유형**: 수치형 (오염물질+업종 명시)  
**기대 동작**: BOD 전체 지역/규모 기준 표 반환. 업종 세차장으로 인식하지만 기준은 지역/규모 기반.

### Q2 (수치형 — 부분정보)
**질의**: "세차장이 청정지역에 있는데 BOD 몇 mg/L까지 가능해?"  
**유형**: 수치형 (오염물질+지역+업종)  
**기대 동작**: BOD+청정지역 2행(large/small) 반환. 수치: 30, 40.

### Q3 (맥락형)
**질의**: "세차장에서 폐수를 배출할 때 허가 절차는 어떻게 되나?"  
**유형**: 맥락형 (절차·행정)  
**기대 동작**: JSON 수치 아님 → RAG로 법령 절차 설명 (배출시설 허가, 측정 주기 등).

### Q4 (음성형)
**질의**: "세차장 폐수의 페놀 기준은 얼마?"  
**유형**: 음성형 (데이터에 없는 항목)  
**기대 동작**: 페놀은 JSON 미등록 → "데이터에 확인되지 않음" 안내, 숫자 추측 금지.

## 식품제조 (음식료 / 도축·수산물·과실채소 가공)

### Q5 (수치형)
**질의**: "식품제조업 TOC 기준은 나지역에서 몇?"  
**유형**: 수치형 (오염물질+지역+업종)  
**기대 동작**: TOC+나지역 2행 반환. 수치: 50(large), 75(small).

### Q6 (수치형 — COD)
**질의**: "음식료 공장의 COD 배출허용기준 알려줘"  
**유형**: 수치형 (COD → TOC 변환)  
**기대 동작**: COD를 TOC로 변환, 안내 메시지 생성 ("현행 법상 TOC로 관리"). TOC 전체 기준 표.

### Q7 (수치형 — 부분정보 미상)
**질의**: "식품공장 SS 기준?"  
**유형**: 수치형 (지역/규모 미상)  
**기대 동작**: SS 전체 8행 반환 ("지역/규모가 명시되지 않아 전체 표시" 안내).

### Q8 (복합형)
**질의**: "2,000㎥ 이상 대규모 식품제조시설의 가지역 T-N 기준과, 그 지역 환경 등급은?"  
**유형**: 복합형 (수치+맥락)  
**기대 동작**: T-N+가지역+large 1행(60); 가지역 정의 문서 인용 (RAG).

## 도금 (도금 / 표면처리)

### Q9 (수치형)
**질의**: "도금업체 특례지역 BOD 제한은?"  
**유형**: 수치형 (오염물질+지역+업종)  
**기대 동작**: BOD+특례지역 2행 반환. 수치: 30(large/small 동일).

### Q10 (맥락형)
**질의**: "도금 시설의 폐수 처리 방법은?"  
**유형**: 맥락형  
**기대 동작**: RAG 검색 → 수질오염방지시설, 처리 흐름 설명.

### Q11 (음성형)
**질의**: "도금 공정의 색도 기준?"  
**유형**: 수치형이지만 색도는 기본 40개 JSON에 미등록  
**기대 동작**: 기준값 없음 → law.go.kr 확인 안내.

## 세탁 (세탁업 / 세탁소)

### Q12 (수치형)
**질의**: "세탁소 BOD T-P 기준을 청정지역에서 소규모 기준으로 알려줘"  
**유형**: 수치형 (복수 오염물질)  
**기대 동작**: BOD 40, T-P 4 (청정지역+small, 각 1행씩).

### Q13 (맥락형)
**질의**: "세탁업 폐수 배출 신고는 어디서?"  
**유형**: 맥락형  
**기대 동작**: RAG → 배출시설 신고 절차, 관할 부서 안내.

### Q14 (음성형 — 규모 기준값)
**질의**: "1,500㎥ 배출량은 세탁 기준 대규모인가?"  
**유형**: 질의는 "1,500"이지만 경계값은 "2,000"  
**기대 동작**: 부분정보 분석 ("2,000㎥ 미만"이면 small, 아니면 large 기준 제시).

## 축산 (축산 / 축산 폐수)

### Q15 (수치형)
**질의**: "축산 폐수의 나지역 소규모 BOD 기준?"  
**유형**: 수치형 (오염물질+지역+규모+업종)  
**기대 동작**: BOD+나지역+small 1행 반환. 수치: 120.

### Q16 (수치형 — 모든 축산 기준 조회)
**질의**: "축산폐수 모든 배출허용기준은?"  
**유형**: 수치형 (오염물질 미지정 → 업종만)  
**기대 동작**: "오염물질이 명시되지 않았으므로" 안내; 결과는 없음 (업종만으로는 수치형으로 보지 않음).

### Q17 (복합형)
**질의**: "축산농가가 가지역에 있는데 SS, T-N, T-P 기준은?"  
**유형**: 복합형 (복수 오염물질+지역)  
**기대 동작**: 3개 오염물질 × 가지역 × (large+small) = 6행 반환.
  - SS: 60(large), 80(small)
  - T-N: 60(large/small)
  - T-P: 8(large/small)

### Q18 (음성형)
**질의**: "축산 시설의 질소산화물 기준?"  
**유형**: 수치형이지만 "질소산화물"(NOx) 미등록  
**기대 동작**: 물환경 배출기준이 아님 → 확인 불가 안내 (대기 기준과 혼동 방지).

## 기타 테스트 케이스

### Q19 (정확성 검증 — COD→TOC)
**질의**: "화학적산소요구량 기준"  
**유형**: 별칭 인식 + 변환  
**기대 동작**: COD→TOC 변환, 안내 메시지, TOC 전체 기준.

### Q20 (정확성 검증 — 부분정보)
**질의**: "청정지역 배출기준?"  
**유형**: 수치형 미아님 (오염물질 무지정)  
**기대 동작**: "오염물질이 명시되지 않았으므로" 안내, matches 비움. RAG로 청정지역 정의.

### Q21 (정확성 검증 — 한글 별칭)
**질의**: "부유물질이 많은데 가지역 기준?"  
**유형**: 수치형 (부유물질=SS 별칭)  
**기대 동작**: SS로 변환, 가지역 2행(60, 80).

### Q22 (정확성 검증 — 모든 지역)
**질의**: "T-P 전체 기준?"  
**유형**: 수치형 (오염물질만, 지역/규모 미지정)  
**기대 동작**: T-P 전체 8행 반환, "지역/규모 미명시" 안내.

### Q23 (정확성 검증 — 특례지역)
**질의**: "특례지역 공업단지의 BOD?"  
**유형**: 수치형 (특례지역)  
**기대 동작**: BOD+특례지역 2행 (30, 30).

### Q24 (정확성 검증 — 큰 숫자)
**질의**: "2,000㎥ 이상 공장의 BOD 청정??"  
**유형**: 수치형 (규모+오염물질+지역, 콤마 포함)  
**기대 동작**: 콤마 인식 (2000 == 2,000), large 선택, BOD+청정+large 1행(30).
